The following are error codes you may experience while importing or exporting:
  • Backup file is "greyed out"
    On your computer, copy the backup file to your local file system. Wait for syncing to complete. Re-add the backup file to your cloud folder. Try importing again.

  • Error Code: 1
    The .zip file you selected is invalid. If you are sure you selected the right one, try exporting it again.

  • Error Code: 2
    The data file inside the .zip wasn't found.

  • Error Code: 3
    The data file inside the .zip is in an invalid format. Try exporting your entries again.

  • Error Code: 4
    The data file inside the .zip is corrupt. Try exporting your entries again.
Warning! If you are repeatedly experiencing these errors, please send a bug report or contact Anglers' Log.
