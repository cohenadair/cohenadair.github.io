The following are error codes you may while using Anglers' Log:
  • Error Code: 5
    The location, bait, species, etc. you attempted to delete didn't work. This normally happens if for some reason the item you're trying to delete has the same name as another item of the same type. Try renaming the item and then deleting it again.
Warning! If you are repeatedly experiencing these errors, please send a bug report or contact Anglers' Log.
